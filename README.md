# Setup

Run the following:

 - `bundle install --without production`
 - `rake db:migrate`
 - `rake db:seed:directors`
 - `rake db:seed:movies`
 - `rake db:seed:actors`
 - `rake db:seed:characters`
 - `rails s`
 - Pull up the [Devise Cheatsheet](https://gist.github.com/raghubetina/9ef4a9ffa4604df74bb5).
